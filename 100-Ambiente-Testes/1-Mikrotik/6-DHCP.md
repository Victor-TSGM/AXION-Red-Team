# Configurando Servidor DHCP 

No menu esquerdo: `IP > DHCP Server > DHCP Setup`
- DHCP Server Interface: LAN
- DHCP Address Space: 192.168.0.0/24
- Gateway for DHCP Network: 192.168.0.1
- Address to Give Out - Informar DHCP Pool: `192.168.0.1-192.168.0.199`
- DNS Servers: `8.8.8.8`
